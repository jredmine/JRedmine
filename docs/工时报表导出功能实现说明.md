# 工时报表导出功能实现说明

## 概述

工时报表导出功能为工时报表模块提供专业的文件导出能力，支持**Excel**、**CSV**、**PDF**三种主流格式，满足不同场景的导出需求。

**开发日期**：2026-01-14  
**开发耗时**：约2小时

## 功能特性

### 支持的导出格式

| 格式 | 特点 | 适用场景 |
|------|------|---------|
| **Excel (.xlsx)** | 格式丰富、样式美观、支持公式 | 数据分析、深度加工 |
| **CSV (.csv)** | 纯文本、轻量级、通用性强 | 数据导入、程序处理 |
| **PDF (.pdf)** | 格式固定、易于分享、专业展示 | 报告存档、打印输出 |

### 支持的报表类型

1. **项目工时报表**
   - 项目基础信息
   - 成员工时分布
   - 活动类型分布

2. **用户工时报表**
   - 用户基础信息
   - 项目工时分布
   - 活动类型分布

3. **时间段工时报表**
   - 时间段统计信息
   - 工时趋势详情

```java
// 标题样式
private CellStyle createTitleStyle(Workbook workbook) {
    CellStyle style = workbook.createCellStyle();

    font.setFontHeightInPoints((short) 14);

private CellStyle createHeaderStyle(Workbook workbook) {
    CellStyle style = workbook.createCellStyle();
    Font font = workbook.createFont();
    font.setBold(true);
    font.setFontHeightInPoints((short) 11);
    style.setFont(font);
    style.setFillForegroundColor(IndexedColors.GREY_25_PERCENT.getIndex());
    style.setFillPattern(FillPatternType.SOLID_FOREGROUND);
    style.setBorderBottom(BorderStyle.THIN);
    // ... 更多边框设置
    return style;
}

// 数据样式
private CellStyle createDataStyle(Workbook workbook) {
    CellStyle style = workbook.createCellStyle();
    style.setBorderBottom(BorderStyle.THIN);
    style.setBorderTop(BorderStyle.THIN);
    style.setBorderLeft(BorderStyle.THIN);
    style.setBorderRight(BorderStyle.THIN);
    return style;
}
```

```java
// 标题
Paragraph title = new Paragraph("Project Time Entry Report")
        .setFontSize(20)
        .setBold()
        .setTextAlignment(TextAlignment.CENTER);

// 表头单元格
private void addPdfHeaderCell(Table table, String text) {
    Cell cell = new Cell().add(new Paragraph(text).setBold());
    cell.setBackgroundColor(ColorConstants.LIGHT_GRAY);
    cell.setTextAlignment(TextAlignment.CENTER);
    table.addHeaderCell(cell);
}

// 信息行
private void addPdfInfoRow(Table table, String label, String value) {
    Cell labelCell = new Cell().add(new Paragraph(label).setBold());
    labelCell.setBackgroundColor(ColorConstants.LIGHT_GRAY);
    table.addCell(labelCell);
    table.addCell(value);
}
```

#### 页脚

自动添加生成时间：

```java
private void addPdfFooter(Document document) {
    document.add(new Paragraph("\n"));
    String footer = "Generated at: " + 
        LocalDateTime.now().format(DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"));
    Paragraph footerPara = new Paragraph(footer)
            .setFontSize(9)
            .setTextAlignment(TextAlignment.RIGHT);
    document.add(footerPara);
}
```

## API接口

### 项目工时报表导出

#### Excel格式
```
GET /api/time-entries/reports/project/export/excel
```

**参数**：
- `projectId` (必填)：项目ID
- `startDate` (可选)：开始日期
- `endDate` (可选)：结束日期
- `year` (可选)：年份
- `month` (可选)：月份

**响应**：
- Content-Type: `application/octet-stream`
- Content-Disposition: `attachment; filename=project_time_report.xlsx`

**示例**：
```bash
curl -X GET "http://localhost:8088/api/time-entries/reports/project/export/excel?projectId=1&year=2026&month=1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -o project_report.xlsx
```

#### CSV格式
```
GET /api/time-entries/reports/project/export/csv
```

**响应**：
- Content-Type: `text/csv`
- Content-Disposition: `attachment; filename=project_time_report.csv`

**示例**：

## 应用场景

### 场景1：项目成本报告

**需求**：生成专业的项目成本报告，提交给项目负责人

**操作**：
```bash
# 导出为PDF，适合打印和存档
curl -X GET "http://localhost:8088/api/time-entries/reports/project/export/pdf?projectId=1&year=2026&month=1" \
  -H "Authorization: Bearer $TOKEN" \
  -o project_cost_report.pdf
```

**价值**：
- 专业格式，易于分享
- 完整的项目工时数据
- 成员和活动类型分布清晰
- 可直接打印

### 场景2：绩效考核数据

**需求**：导出员工工时数据，用于Excel中进一步分析

**操作**：
```bash

```

**价值**：
- 支持Excel公式和图表
- 可以进行数据透视
- 便于计算和对比
- 保留样式和格式

### 场景3：数据备份

**需求**：定期备份工时数据，用于长期存档

**操作**：
```bash
# 导出为CSV，轻量级、长期保存
curl -X GET "http://localhost:8088/api/time-entries/reports/period/export/csv?year=2026" \
  -H "Authorization: Bearer $TOKEN" \
  -o time_entries_2026_backup.csv
```

**价值**：
- 文件小，易于存储
- 纯文本，永不过时
- 易于批量处理
- 可导入其他系统

### 场景4：自动化报表

**需求**：每月自动生成工时报表并邮件发送

**实现思路**：
```bash
#!/bin/bash
# 月度工时报表自动生成脚本

TOKEN="your_token_here"
YEAR=$(date +%Y)
MONTH=$(date +%-m)

# 生成PDF报表
for project_id in 1 2 3; do
    curl -X GET "http://localhost:8088/api/time-entries/reports/project/export/pdf?projectId=${project_id}&year=${YEAR}&month=${MONTH}" \
      -H "Authorization: Bearer ${TOKEN}" \
      -o "project_${project_id}_${YEAR}_${MONTH}.pdf"
done

# 发送邮件（伪代码）
send_email --attachment project_*.pdf --to manager@company.com
```

### 场景5：跨系统数据交换

**需求**：将工时数据导出后导入到其他系统

**操作**：
```bash
# 导出为CSV，易于程序处理
curl -X GET "http://localhost:8088/api/time-entries/reports/period/export/csv?periodType=month&year=2026" \
  -H "Authorization: Bearer $TOKEN" \
  -o time_data_export.csv

python import_to_other_system.py time_data_export.csv
```

## 性能优化

### 1. 内存管理

- 使用`ByteArrayOutputStream`直接在内存中生成文件
- 及时关闭资源（使用try-with-resources）
- 避免创建大量临时对象

### 2. 样式复用

```java
// 创建样式一次，多次使用
CellStyle headerStyle = createHeaderStyle(workbook);
CellStyle dataStyle = createDataStyle(workbook);

// 为多个单元格应用相同样式
for (int i = 0; i < 5; i++) {
    row.getCell(i).setCellStyle(dataStyle);
}
```

### 3. 批量写入

- 先构建完整的数据结构
- 一次性写入输出流
- 减少IO操作次数

### 4. 数据复用

```java
// 复用报表生成的数据
TimeEntryProjectReportDTO report = timeEntryService.generateProjectReport(request);

// 直接使用报表数据进行导出
// 避免重复查询数据库
```

## 错误处理

### 异常捕获

```java
try {
    // 导出逻辑
} catch (Exception e) {
    log.error("导出项目工时报表Excel失败", e);
    throw new BusinessException("导出Excel失败: " + e.getMessage());
}
```

### 常见错误

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| 文件无法打开 | 格式错误 | 检查导出逻辑 |
| 中文乱码 | 编码问题 | 添加UTF-8 BOM |
| 样式缺失 | 样式未应用 | 检查样式设置代码 |
| 内存溢出 | 数据量过大 | 分批导出或增加内存 |
